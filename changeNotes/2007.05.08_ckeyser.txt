Description:

1) Species codes used in the western variants have been standardized. For
     example, other softwoods was "OS" in some variants and "OC" in others.
     It is now "OS" in all variants.  This change will allow users in the west
     to run their data sets through different variants if they want, without
     changing the input datasets.  Changes by variant are as follows:

2) A species translator was added to all western variants.  This will allow
     any recognized western FVS Alpha code, FIA code, or USDA Plants symbol
     to be translated to the most appropriate species in a given variant during
     data input.  Species mapping was approved by regional FVS contacts in
     April 2007.  The table below contains the species mapping of input FVS
     alpha codes (ALFA column), FIA codes (FIA column), and plants symbols (PLNT
     column) by FVS variant (2-digit code).


Impact on users:  Users may enter any recognized species code in the table below 
     and FVS will map it to the most appropriate species for the variant in use.
